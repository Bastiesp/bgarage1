# BGarage CRM SaaS Taller

Primera versión funcional para Render: login seguro, Kanban, presupuestos, tarjetas de aceite, reparaciones con fotos Cloudinary, costos/utilidad y gestión comercial.

## Render
- Build Command: `npm install`
- Start Command: `npm start`
- Variables: `MONGODB_URI`, `JWT_SECRET`, `ADMIN_EMAIL`, opcionales `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET`.

## Primer usuario
Regístrate desde `/register.html`. Si el email coincide con `ADMIN_EMAIL`, queda como admin.
